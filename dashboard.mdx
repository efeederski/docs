---
title: "Beranda"
description: "Halaman Beranda pada aplikasi eFeeder berfungsi sebagai dashboard monitoring yang menampilkan informasi perbandingan data antara Sistem Informasi Akademik (SIA / eAkademik) dengan Neo Feeder PDDIKTI."
---

Melalui halaman **Beranda**, operator dapat mengetahui **kesesuaian jumlah data**, **potensi selisih data**, serta menentukan **tindak lanjut sinkronisasi atau perbaikan data** sebelum proses pelaporan dilakukan.

Halaman Beranda digunakan untuk :

<Columns cols={2}>
  <Column>
    <Card icon="calculator" title="Jumlah Data">
      Menampilkan perbandingan jumlah data antara **SIA (eAkademik)** dan **Neo Feeder**
    </Card>
  </Column>
  <Column>
    <Card icon="arrows-from-line" title="Selisih Data">
      Mengetahui adanya **selisih data** antara **SIA (eAkademik)** dan **Neo Feeder**
    </Card>
  </Column>
</Columns>

<Columns cols={2}>
  <Column>
    <Card icon="award" title="Kontrol Kualitas Pelaporan">
      Membantu operator melakukan **kontrol kualitas data pelaporan**
    </Card>
  </Column>
  <Column>
    <Card icon="arrows-repeat-1" title="Persiapan Awal Pelaporan">
      Menjadi titik awal sebelum melakukan sinkronisasi atau perbaikan data
    </Card>
  </Column>
</Columns>

<Info>
  Di bagian atas halaman Beranda tersedia **filter pencarian** yang digunakan untuk menentukan data yang akan ditampilkan.
</Info>

### Komponen Filter

<Columns cols={3}>
  <Column>
    <Card title="Semester">
      Digunakan untuk memilih semester pelaporan yang akan dianalisis.
    </Card>
  </Column>
  <Column>
    <Card title="Program Studi">
      Digunakan untuk memfilter data berdasarkan program studi tertentu.
    </Card>
  </Column>
  <Column>
    <Card title="Jenis Data">
      Digunakan untuk memilih jenis data akademik yang ingin ditampilkan.
    </Card>
  </Column>
</Columns>

### Tombol Aksi

<Columns cols={2}>
  <Column>
    <Card title="Tampilkan">
      Menampilkan data perbandingan sesuai filter yang dipilih.
    </Card>
  </Column>
  <Column>
    <Card title="Reset">
      Mengosongkan seluruh filter data pencarian.
    </Card>
  </Column>
</Columns>

> **Catatan :**
>
> Seluruh field filter wajib diisi sebelum data dapat ditampilkan.

Setelah filter dipilih dan tombol **Tampilkan** diklik, sistem akan menampilkan **panel perbandingan data SIA dengan Neo Feeder**.

Jenis data yang ditampilkan meliputi:

- Kurikulum
- Mata Kuliah Kurikulum
- Mahasiswa
- Kelas
- KRS
- KRS Nilai
- Nilai Transfer

<Info>
  Setiap panel menampilkan informasi ringkas jumlah data dan status perbandingan.
</Info>

### Informasi perbandingan pada setiap panel

<Card icon="graduation-cap" title="Jumlah Data di SIA (eAkademik)">
  Menunjukkan total data yang tercatat di sistem akademik kampus.
</Card>

<Card icon="solar-system" title="Jumlah Data di Neo Feeder">
  Menunjukkan total data yang sudah tercatat di Neo Feeder PDDIKTI.
</Card>

<Card icon="calculator-simple" title="Selisih Data">
  Menunjukkan perbedaan jumlah data antara SIA dan Neo Feeder.
</Card>

<Warning>
  Selisih dihitung dari perbandingan jumlah data di SIA dengan jumlah data di Neo Feeder.
</Warning>

### Keterangan Selisih Data

Di bagian bawah halaman Beranda tersedia **keterangan selisih data** sebagai panduan interpretasi hasil perbandingan. Keterangan tersebut menjelaskan bahwa:

<Card icon="equals" title="Tidak Ada Selisih">
  Jumlah data di SIA sama dengan jumlah data di Neo Feeder.
</Card>

<Card icon="greater-than-equal" title="Data di SIA lebih banyak">
  Terdapat data di sistem akademik yang belum tersinkron ke Neo Feeder.
</Card>

<Card icon="less-than-equal" title="Data di SIA lebih sedikit">
  Terdapat data di Neo Feeder yang tidak atau belum tercatat di SIA.
</Card>
